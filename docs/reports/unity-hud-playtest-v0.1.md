# Unity HUD 플레이테스트 기록 v0.1

> 상태: **현행** · 최종 갱신: 2026-07-16 · 커밋 `f1cf138` 기준

## 1. 목적과 방법

Unity HUD와 영웅 조작에서 발견된 사용성 문제를 구현 계획과 분리해 보존하는 정성 근거다.
2026-07-16 사용자 플레이 관찰과 피드백을 원문 의미가 바뀌지 않도록 정리했다. 이번 기록에는
화면 녹화, 클릭 로그, 과업 성공률 같은 정량 측정이 없으므로 발생 빈도와 심각도를 수치로 확정하지
않는다. 수정 후 같은 절차로 재검증 결과를 별도 버전 리포트에 남긴다.

## 2. 관찰

| ID | 관찰된 문제 | 기대 동작 | 분류 |
|---|---|---|---|
| F1 | 영웅 경험치 구매를 클릭하다 레벨업하면 증강 카드가 즉시 같은 위치에 나타나 원치 않는 카드를 누른다. | backdrop은 즉시 입력을 차단하고 카드는 1초 뒤 활성화된다. | 입력 안전 |
| F2 | 보스 소환 쿨타임이 하단 전폭 bar라 버튼별 게임 쿨타임처럼 읽히지 않는다. | 각 해금 버튼 위에 시계방향 radial 진행과 남은 초가 표시된다. | 상태 가독성 |
| F3 | 영웅 사망 시 남은 부활 시간이 잘 보이지 않는다. | 선택 상태와 무관하게 부활 시간과 부활 위치를 즉시 찾는다. | 전투 피드백 |

## 3. 재현 절차

### F1 — 증강 오클릭

1. 다음 XP 구매로 증강 지급 레벨이 되는 상태를 만든다.
2. XP 구매 버튼을 빠르게 연속 클릭한다.
3. 증강 선택 화면이 나타난 직후 카드가 선택되는지 확인한다.

### F2~F5 — 정보 탐색

1. 보스를 소환해 쿨타임 상태를 만든다.
2. 영웅을 사망시키고 다른 타워 또는 빈 타일을 선택한다.
3. 서로 다른 증강을 여러 장 획득한다.
4. 영웅과 공격 간격이 다른 타워를 차례로 선택한다.
5. 설명 없이 쿨타임, 부활 시간, 보유 증강, 공격속도를 찾고 의미를 말하게 한다.

### F6 — 영웅 이동

1. 길 타일의 중앙과 가장자리를 각각 우클릭한다.
2. 직선 구간과 90도 코너를 가로지르는 목적지를 지정한다.
3. 실제 도착점, 이동 궤적, 목적지·경로 표시가 클릭 의도와 일치하는지 확인한다.
