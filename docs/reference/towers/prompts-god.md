# GOD 티어 프롬프트 (11종 · 독립 영웅)

> 근거: [컨셉 §7](../../design/tower-dot-modeling-concept.md).
> 계열의 연장이 **아니다**. 각 병과를 대표하는 **일점 전력**이라 독립 실루엣 + 발판 강림 후광 +
> 왕관/오라로 "타워가 아니라 강림한 존재"로 그린다. 병과색은 **강조 트림·발광**으로만(전신 tint 금지).
> 초기 풀(보스 5처치 이하)과 확장 풀(6처치↑)을 구분한다. 자체 완결형 프롬프트 — 하나씩 붙여넣는다.

---

## GOD 공용 문법 (모든 프롬프트에 포함됨)

- 개별 1인(또는 1기) **히어로 컷**, 정면 3/4 위풍 포즈.
- 발밑에 **부드러운 강림 후광**(soft glowing descent halo) — GOD만의 표식.
- 계열 타워보다 **크고 아이코닉**하게. 병과색은 트림·발광 액센트로만.
- 배경 무지 회색, 후광 외 그림자 없음, 글자·숫자·라벨 없음.

**병과 액센트색:** 정규군 `#6d86ab` · 포병 `#bf7a3a` · 마법대 `#9a6ea6` · 소환대 `#8a9a5b`.
