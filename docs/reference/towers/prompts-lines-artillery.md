# 포병 계열 프롬프트 (race 1 · 청동 `#bf7a3a`)

> 근거: [컨셉 §4](../../design/tower-dot-modeling-concept.md).
> **병사보다 병기가 주역** — 조작수는 크기 대비로 규모를 읽히는 보조.
> **두 계열은 실루엣 방향으로 즉시 구분:** 투석기 = **세로로 던지는 팔**(곡사), 노포 = **가로 볼트 축**(직사).
> 각 프롬프트는 자체 완결형이다.

---

## 1. 투석기 계열 (power)

`artillery-small-catapult → artillery-mangonel → artillery-trebuchet → artillery-megalith-catapult`
(소형 투석기 → 망고넬 → 트레뷰셋 → 거석 투석기)

**계열 표식:** **세로 방향 던지는 팔** + 돌 탄. 팔·프레임·평형추가 커지며 규모로 상승을 읽힌다.

```
A horizontal siege-engine evolution sheet: ONE catapult line, four full war machines in a single
row, tier 1 to tier 4 from left to right, side 3/4 view. The SAME weapon family, progressively
larger and more reinforced. A tiny chibi operator crew stands beside each to read scale.

STYLE: chibi low-poly fantasy war machine, chunky rounded proportions, angular low-poly faceting
with soft cel shading, bold clean readable silhouette. Muted material palette: brown timber, aged
bronze fittings, thick rope, dark iron. Even soft studio lighting from upper-left. Painted
game-asset concept style.

FRAME: flat neutral mid-gray background, NO cast shadow, NO ground line, full machine in frame,
identical ground baseline and camera height for every machine, evenly spaced. Absolutely no text,
no letters, no numbers, no UI, no icons, no watermark.

Machine 1 (Lv1, humblest): a small wooden throwing arm with a twisted-rope torsion bundle and a
single small rock, no wheels, one small operator.
Machine 2 (Lv2): a larger arm with bronze reinforcement and a leather sling pouch, on wooden wheels.
Machine 3 (Lv3): a tall A-frame trebuchet with a long throwing arm, a counterweight box, bronze hinges.
Machine 4 (Lv4, iconic and massive): a giant double-frame catapult with a chained counterweight,
a huge boulder projectile, a bronze winch, two operators, a small pennant flag.

CONSISTENCY: every machine reads as a VERTICAL throwing arm that lobs stone (indirect fire). Scale
and frame density grow tier by tier. Use the accent color ONLY on the pennant / bronze trim / rope
wrap: bronze #bf7a3a. Materials stay in the fixed palette; do not tint the whole machine.
```

## 2. 노포(발리스타) 계열 (power)

`artillery-small-ballista → artillery-ballista → artillery-heavy-ballista → artillery-repeating-ballista`
(소형 노포 → 발리스타 → 중발리스타 → 연발 발리스타)

**계열 표식:** **수평 활대 + 볼트**(직사). 활대가 두꺼워지고 Lv4는 회전 탄창(리볼버식)으로 다연장화.

```
A horizontal siege-engine evolution sheet: ONE ballista line, four full war machines in a single
row, tier 1 to tier 4 from left to right, side 3/4 view. The SAME weapon family, progressively
heavier and more reinforced. A tiny chibi operator crew stands beside each to read scale.

STYLE: chibi low-poly fantasy war machine, chunky rounded proportions, angular low-poly faceting
with soft cel shading, bold clean readable silhouette. Muted material palette: brown timber, aged
bronze fittings, thick rope, dark iron. Even soft studio lighting from upper-left. Painted
game-asset concept style.

FRAME: flat neutral mid-gray background, NO cast shadow, NO ground line, full machine in frame,
identical ground baseline and camera height for every machine, evenly spaced. Absolutely no text,
no letters, no numbers, no UI, no icons, no watermark.

Machine 1 (Lv1, humblest): a small wooden bow-limb ballista with a short bolt, on a simple tripod.
Machine 2 (Lv2): bronze-reinforced bow limbs, a bolt channel, a crank winch, a small pavise shield.
Machine 3 (Lv3): double thick bow limbs, a large mounted bolt, an iron frame.
Machine 4 (Lv4, iconic): a revolver-style rotating bolt magazine, bronze gears, an operator — a
multi-shot repeating ballista.

CONSISTENCY: every machine reads as a HORIZONTAL bow axis firing a bolt (direct fire) — clearly
distinct from the vertical catapult arm. The limbs thicken and the Lv4 gains a rotating magazine.
Use the accent color ONLY on the pavise / bronze trim / rope: bronze #bf7a3a. Materials stay in the
fixed palette; do not tint the whole machine.
```
