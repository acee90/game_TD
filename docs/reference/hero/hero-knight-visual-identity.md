# 방랑기사(Knight) 비주얼 아이덴티티

> 상태: **현행** · 최종 갱신: 2026-07-23
> 근거 이미지: [`hero-knight-concept-v3.png`](hero-knight-concept-v3.png) (기준 컨셉아트)
> 대조: [`hero-knight-chibi-low-poly.png`](hero-knight-chibi-low-poly.png) (스타일 변형 사례)
> 목적: **"캐릭터 고유 외형"과 "예술적 화풍"을 분리 기록**해서, 화풍이 바뀌어도
> (컨셉아트 → 치비 → 도트) 같은 캐릭터로 읽히도록 일관성 기준을 고정한다.
> 파이프라인: [컨셉아트 먼저 → 승인 → 도트화](../../design/tower-dot-modeling-concept.md).
> 이 문서는 이름·형태(플레이버)만 다룬다 — 밸런스·태그·스탯과 무관하다.

---

## 왜 분리하는가

같은 캐릭터를 여러 화풍으로 그리게 된다(사실적 컨셉아트 · 치비 저폴리 · 도트).
화풍이 바뀔 때 **바뀌어도 되는 것(=화풍)** 과 **절대 바뀌면 안 되는 것(=고유 외형)** 을
구분하지 않으면, 리사이즈·재렌더 때마다 다른 캐릭터가 나온다.

- **고유 외형특징** = 어떤 화풍으로 그려도 유지해야 하는 캐릭터의 정체성.
- **예술적 화풍** = 이 이미지 한 장의 렌더링 방식. 화풍을 바꾸는 게 목적일 땐 통째로 교체 가능.

기준 이미지(v3)와 치비 변형을 비교하면 이 경계가 보인다. 치비는 화풍(비율·렌더)만
바꾸면서 고유 외형(투구 실루엣 · 보라 깃털 · 은빛 판금 · 남색 망토)은 유지했고,
저해상도에서 담기 어려운 세부(켈트 매듭 · 사선 벨트)는 버렸다.

---

## A. 캐릭터 고유 외형특징 (identity — 화풍 불문 유지)

일관성 우선순위를 3단계로 나눈다. **T1은 반드시**, T2는 가능하면, T3은 여유 있을 때.

### T1 — 실루엣/식별 (도트·치비에서도 반드시 유지)

| 요소 | 묘사 | 비고 |
|---|---|---|
| **닫힌 대투구** | 얼굴 전체를 덮는 클로즈드 헬름(바시넷 계열). 둥근 두정부 + 각진 안면. | 캐릭터의 **1순위 실루엣**. 맨얼굴 없음. |
| **보라/자홍 깃털 장식** | 투구 정수리에 **단일 직립 깃털**, 끝이 갈라짐. 채도 높은 자주색. | **유일한 고채도 포인트 컬러** — 회색 갑옷 속에서 캐릭터를 식별시키는 시그니처. 절대 제거 금지. |
| **가로 시야 슬릿 1줄** | 눈 위치에 수평 관측 틈 하나. | |
| **세로 통기 슬릿** | 안면 하단에 세로 홈 3~4개. | 저해상도에선 2~3개로 축약 가능. |
| **은빛 풀플레이트** | 전신 강철 판금. 무채색 실버·차콜. | |
| **남색(네이비) 롱 케이프** | 등에서 발목까지 내려오는 짙은 남색 망토. | 실루엣 하단 부피를 만드는 핵심. |
| **한손 직검** | 곧은 양날 브로드소드/롱소드를 오른손에 세워 듦. | |

### T2 — 재질/컬러 (여유 픽셀 있으면)

| 요소 | 묘사 |
|---|---|
| **남색 천 액센트** | 목을 감은 스카프/고젯 랩, 판금 아래 남색 서코트·탭바드. 갑옷 실버와 대비. |
| **갈색 가죽 벨트류** | 허리 벨트 + 가슴을 가로지르는 **사선 발드릭(멜빵)**, 오른쪽 엉덩이 **가죽 파우치**. |
| **투구 측면 리벳 디스크** | 귀 위치의 둥근 피벗 볼트(원형 판). |
| **너덜한(tattered) 밑단** | 케이프와 탭바드 하단이 뾰족뾰족 찢긴 가장자리. |
| **비늘/해머드 텍스처** | 판금 표면의 물고기비늘·망치자국 질감. |

### T3 — 문양/디테일 (컨셉아트·고해상도에서만)

| 요소 | 묘사 |
|---|---|
| **켈트 매듭 문양** | 가슴 스트랩 버클 · 검 폼멜 · 케이프 하단 모서리(금색 자수)에 동일한 **사엽 매듭(quatrefoil knot)** 반복. 캐릭터의 문장(紋) 역할. |
| **검 손잡이 감기** | 어두운 남색으로 감은 그립 + 십자 가드. |
| **금색 켈트 트림** | 남색 케이프 밑단 모서리의 금색 매듭 자수. |

> **컬러 팔레트 요약(고유):** 은빛 강철 + 남색(네이비) 천 + 갈색 가죽 + **자주색 깃털(포인트)** + 금색 켈트 액센트.

## B. 예술적 화풍 (style — 화풍 교체 시 통째로 대체)

기준 이미지 v3의 렌더링 방식. 다른 화풍(치비·도트)으로 갈 때는 아래를 버리고 새 화풍 규칙으로 대체한다.

| 항목 | v3 컨셉아트 |
|---|---|
| **장르/기법** | 세미리얼리즘 판타지 캐릭터 컨셉 일러스트. 페인터리 디지털 채색. |
| **비율** | 사실적 성인 영웅 체형(약 7~8등신). |
| **채색** | 무채색 중심의 **탁하고 저채도** 팔레트. 사실적 금속 반사·하이라이트. |
| **디테일 밀도** | 높음 — 비늘 갑옷·해머드 메탈·가죽 스티치까지 묘사. |
| **배경/구도** | 따뜻한 회색 무지 배경(컨셉 시트). **전신 정면 1컷 + 투구 클로즈업 콜아웃**. |
| **조명/그림자** | 부드러운 앰비언트. 발밑에 옅은 접지 그림자. |
| **라인** | 뚜렷한 외곽선 없이 명암으로 형태를 잡음. |

### 대조 — 치비 저폴리 화풍 ([hero-knight-chibi-low-poly.png](hero-knight-chibi-low-poly.png))

| 항목 | 치비 저폴리 |
|---|---|
| **비율** | 슈퍼디폼드(약 2등신), 머리·투구 과장. |
| **기법** | 로우폴리 3D 룩, 각진 패싯 셰이딩. |
| **구도** | 앞면 + 뒷면 턴어라운드. |
| **유지된 고유특징(T1)** | 투구 실루엣 · 자주 깃털 · 은빛 판금 · 남색 케이프 · 갈색 벨트 · 직검. ✅ |
| **버려진 특징(T3)** | 켈트 매듭 문양 · 사선 발드릭 · 세밀 텍스처. (저폴리에서 담기 어려움) |

→ **결론:** 화풍이 완전히 달라져도 **T1 목록이 유지되면 같은 캐릭터로 읽힌다.**
도트화할 때도 이 T1을 우선 보존하고, T3는 과감히 생략한다.

## 도트화 체크리스트 (일관성 유지용)

새 화풍/해상도로 이 캐릭터를 그릴 때 아래를 확인한다.

- [ ] 닫힌 대투구 실루엣(맨얼굴 없음)
- [ ] **자주색 직립 깃털** — 유일한 포인트 컬러, 반드시 존재
- [ ] 가로 시야 슬릿 + 세로 통기 슬릿
- [ ] 은빛 판금 + 남색 롱 케이프의 실버↔네이비 대비
- [ ] 오른손 직검
- [ ] (여유 시) 갈색 벨트·파우치, 너덜한 케이프 밑단
- [ ] (고해상도만) 켈트 사엽 매듭 문양
