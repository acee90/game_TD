# 랜덤 합성 타워디펜스 — 게임 디자인 초안 v0.3

- 가제: **머지 가디언즈** (임시)
- 플랫폼: 모바일 1순위, PC(Steam) 크로스플랫폼 목표
- 코어: 싱글 로그라이트 → 협동/경쟁 멀티 확장
- 랜덤성: 랜타디급 + 명시적 복구 수단 + 운 주도 고점
- v0.3 변경: **웨이브 드래프트 전면 삭제**(고정 40라운드 회귀),
  **클리어율 ≤1%의 극악 난이도 선언**, **상인 고용제 전환**(운영 레버화),
  운영 레버 패키지 신설. 기존 시뮬 수치는 드래프트 전제라 전부 폐기 — 재검증 대기.

---

## 1. 한 줄 컨셉

"고정된 40라운드의 게이건틀릿. 100판에 1판만 정상에 도달한다.
운이 겹치고(초월·시너지) 운영이 무결점(미션 체인·상인 경제)인 판만 클리어라는
사건을 만들며, 나머지 99판은 '몇 라운드까지 갔는가'의 기록을 남긴다."

## 2. 디자인 필러

1. **운은 크게, 복구는 명확하게.** 랜타디급 분산 + 유한한 복구 밸브.
2. **고점 = 클리어 그 자체.** 클리어율 ≤1%. 평소의 성과는 도달 라운드이고,
   운의 중첩(초월 강림, 시너지 고리, 상인 경제 개화)과 무결점 운영이 모두 겹친
   판에서만 클리어가 터진다. 그 사건의 기억(혹은 목격담)이 재시작을 누르게 한다.
   (원조 랜타디: 클리어까지 10~100시간 / 개인운빨디펜스: 공방 클리어율 1%대 —
   검증된 스테디셀러 포지션)
3. **선택은 베팅이 아니라 운영에서.** 웨이브는 고정 — 위험을 "구매"하는 압박 없음.
   플레이어의 선택은 자원 운영(§3.5 운영 레버)에 집중된다.
4. **고점이 떠도 운용은 실력.** 하이롤 유닛의 배치·계보·시너지 운용이 결과를 가른다.
5. **인게임 외 성장은 선택지 확장만.** 공격력 곱연산형 계정 성장 배제.

## 3. 런 구조 (v0.3 전면 개정)

### 3.1 기본 골격
- **총 40라운드 고정 시퀀스.** 10/20/30 보스, 40 최종보스. 랜타디 원형 회귀.
- 몹은 경로를 따라 진행. 놓친 몹은 오염 게이지를 올린다.
- 세션: 대부분의 판은 중도 사망으로 끝나므로 평균 세션은 짧고(10~20분),
  깊이 간 판만 30분+. "어디서 죽었는가"가 그 판의 이야기.

### 3.2 고정 웨이브 시퀀스 (드래프트 삭제)
- 웨이브 드래프트는 v0.3에서 **전면 삭제**. 이유: 위험 선택이 고점의 전제가 되는
  순간 압박이 되고, 선택 피로도 유발. 고점은 운과 운영이 정한다.
- 40라운드 시퀀스에 특수 웨이브를 **고정 위치 배치**: 신속(감속/스턴 우대),
  중갑(단일 고딜 우대), 쇄도(광역 우대), 정예, 회복 등.
  "17라운드는 신속"을 외우는 것 자체가 랜타디식 지식층 콘텐츠.
- 특수 웨이브는 소프트체크 유지: 대응 아키타입이 없어도 스탯으로 돌파 가능하되
  불리할 뿐, 특정 타워를 필수화하지 않는다 (조합 강제 최소화 철학 유지).
- 배당 몰수·스트릭·에스컬레이션은 드래프트와 함께 삭제.

### 3.3 영토 확장 — 운영 레버 (유지)
- 5라운드마다 봉인 타일 3개 중 1개 선택 해방: 타워 슬롯 + 특수 지형
  (도박 제단 / 골드 광맥 / 공명석 등) vs 적 진입로 +1의 트레이드.

### 3.4 오염 게이지 (라이프 대체, 유지)
- 놓친 몹 1기 = 오염 +1% (특수 웨이브는 가중). 100% = 패배.
- 25/50/75% 도달 시 페널티만: 오염 지형 (필드 DPS 영구 -3%씩).

### 3.5 운영 레버 패키지 (드래프트가 담당하던 "선택"의 이관처)
| 레버 | 내용 | 계보 |
|---|---|---|
| **상인 고용** | 뽑기 풀에서 제외. 골드로 직접 고용(선택). §5.4 | Legion TD 일꾼 경제 |
| **개인 보스미션** | 쿨타임제 도전 몹, 3단계. **성공 후 쿨 5라운드(약 3분 20초), 실패 시 오염 +10·쿨 3라운드**. 도전 타이밍 계산 = 핵심 실력 요소, 빠를수록 부유 | 랜타디 개인미션 (원작 6분 쿨을 세션 27분에 맞게 단축) |
| 선택권 화폐 | 보스시민(T3 선택권)·만능 토큰의 사용처·타이밍 | 랜타디 보스시민 / 스타2판 테라진 |
| 배짱 미션 | 필드 적 25기 누적 유지(옵트인, 오염 감수) → 골드 | 랜타디 절벽컨의 옵트인화 |
| 영토 확장 | 3택1 타일 + 진입로 트레이드 | 고유 |
| 계보 배분 | 골드를 어느 원소 계보에 직접 쓰는가 = 빌드 선언 | 랜타디 업글 계보 |
| 리롤·매각 | 합성 결과 리롤(라운드 1회), 반값 매각 재투자 | 원랜디 리롤 |

## 4. 고점 시스템 (운의 중첩)

### 4.1 공명과 초월 (유지)
- 동일 신화 2개 = 공명(스탯 점프), 합성 시 6티어 초월체(확정).
- 클리어율 1% 체제에서 초월은 사실상 **클리어의 전제 조건**에 가깝게 배치.
  실측 (v0.3 시뮬): 등장률 15~17% = **6~7판에 1번**, 최초 등장 중위 R19~21 —
  로스터 7종화 + 긴축 경제로 촉매 없이 잭팟 포지션 달성 (기존 빈도 과다 이슈 해소).
  랜타디의 "업적이 유기적으로 연결되어 하나 놓치면 클리어 무산" 구조 계승.
- 후반 도착 초월은 심연 진입 펌프로 활용 (기존 분석 유지).

### 4.2 상인 경제의 개화 (§5.4)
- 고용은 선택(운영)이지만, 고용한 상인이 합성·업그레이드로 얼마나 빨리 크는가와
  황금 상인(고용 시 낮은 확률로 등장, 생산량 2배) 여부는 운 —
  운영과 운이 한 레버 안에서 겹치는 구조.

### 4.3 기록 카드와 도달 지표
- 성과의 기본 단위 = **도달 라운드**. 런 종료 시 기록 카드: 도달 라운드 /
  사망 원인 웨이브 / 초월·미션 달성 / 수입 그래프. 공유 가능.
- **첫 클리어 명예**: 클리어는 사건이므로 전당(클리어자 명단·기록 카드 영구 전시),
  일일 시드에서의 클리어는 별도 표기. (메운디 사례: 첫 클리어자에 대한 관심이
  그 자체로 방송·커뮤니티 콘텐츠가 됨)
- 심연: 클리어자 전용 무한 구간 — 1% 체제에서는 극소수의 명예 공간.

## 5. 타워 시스템

### 5.1 티어와 합성 (유지)
| 티어 | 명칭(안) | 종수 | 획득 | 매각 |
|---|---|---|---|---|
| T1 | 일반 | 7 (상인 제외) | 뽑기 60골드 (7종 균등) | 30 |
| T2~T4 | 희귀/영웅/전설 | 각 7 | 같은 것 ×2 → 상위 랜덤 | 60/120/240 |
| T5 | 신화 | 7 | 〃 | 매각 불가 |
| T6 | 초월 | 8 (상인 포함) | 동일 신화 ×2 (확정) | 매각 불가 |
- 상인이 뽑기 풀에서 빠지며 전투 로스터는 티어당 7종. 동일 쌍 확률이 소폭
  상승하므로 초월 빈도 재검증 필요 (시뮬 대기).
- 황금 꽝(T4 고매각가 1종) 유지.

### 5.2 업그레이드 계보 / 5.3 로스터 매트릭스 (유지 + 차등화 확정)
- 4원소 × 딜/유틸 아키타입, 계보 단위 업그레이드, 물리 창발 시너지(§5.5) 모두 유지.
- **원소별 차등 업그레이드 확정** (레벨당 DPS ×1.03 등가, 상세: 타워스킬시트 §10):
  화염 = 화력(피해) / 번개 = 속도(피해+공속) / 대지 = 파괴(피해+방깎량) /
  냉기 = 지배(피해+감속반경+상인 킬골드). 원소 투자 = 빌드 성향 선언의 수치화.
- 매트릭스의 "검사하는 웨이브 카드" 열은 "우대받는 특수 웨이브"로 의미 변경
  (드래프트 카드 → 고정 시퀀스 내 특수 웨이브).

### 5.4 상인 — 뽑기 등장 + T1 전액 환급 (v0.3.1 개정, "획득 = 운, 유지 = 운영")
- **획득**: T1 뽑기 풀 복귀 (8종 균등, 12.5%). 뽑고 싶을 때 뽑을 수는 없다.
- **T1 전액 환급 (60골드)**: 원치 않으면 무손실 매각 — 상인만 떠서 라운드를 강제로
  지는 브릭 없음 (랜타디 리버의 무손실 변형). 합성한 순간부터 표준 매각 50%,
  T5+ 매각 불가 = 커밋 구조.
- **기능**: 몬스터 처치당 골드 (T1 +1/킬 → 티어당 ×2, 황금왕 +32/킬).
  스톨 역인센티브 없음, 누출 = 수입 감소, 쇄도 웨이브 = 수확제, 킬 코인 팝 피드백.
  필드 슬롯을 차지 — "전투 타워 vs 미래 수입"의 명시적 기회비용.
- **성장·오염 방지**: 상인은 종류가 없어 같은 티어 2기 = 확정 상위. 단, 상인은
  T1 뽑기에서만 등장하고 전투 합성 결과 풀에는 없음. 초반 첫 4회 중복 방지 보정은
  전투 7종 한정 (시작 라인업에 상인 배제).
- **운영의 재미**: 고용 버튼 대신 "판다 vs 킵한다"가 상인이 뜰 때마다의 무비용
  명시적 결정. 황금 변형(10%, 킬골드 ×2)이 킵 판단을 흔드는 운·운영 교차점.
  황금왕(32기 상당)은 "상인이 잘 뜬 판 + 전부 킵한 결단"의 합작 — 운 주도 고점
  철학 정합. 신체계 수치는 재검증 대기 (등장률·환급·킵 전략 균형).
- 실측 (v0.3 V4, 적응형 정책 = 전투 잉여 있을 때만 고용): 상인 투자량은 클리어
  확률을 바꾸지 않고(전 전략 ≈1%) **리스크의 시점을 바꾼다** — 무상인은 중반 안정·
  늦은 붕괴, 경제올인(최대 4기)은 R20 조기 시험(생존 35%)을 통과하면 경제 우위.
  플레이 스타일 선택이 성립.
- 계보: Legion TD의 일꾼(경제 투자 vs 방어 투자)을 합성 디펜스 문법으로 이식.

### 5.5 원소 시너지 (v0.3.1: 두 시너지를 "의존 페어"로 승격)
- 밀집(냉→화) / 존(냉→대) / 고정(번→대) / 배율(대→번) / 수확(화→상인) 시너지,
  CC 감쇠 규칙, 원소 순환 고리 전부 유지.
- **의존 페어 승격**: 배율 시너지 → "속사수는 방깎 필수" (공속 비례로 아머 피해가
  커지는 타격당 감산 모델), 밀집 시너지 → "화포는 감속 필수" (광역 타격 수가
  감속량×존 커버리지의 연속 함수). 단, 서포트는 체크박스가 아니다 —
  **서포트 수치가 서포트 원소 계보 레벨에 스케일**하므로(타워스킬시트 §0 전역 규칙),
  메인딜 덱은 크로스 원소 이중 계보 투자가 강제되고 골드 투자 비율이 빌드 결정이 됨.
- **원소 이중 역할 원칙 (v0.3.2)**: 모든 원소는 파트너 역할과 자기 메인 플랜을 겸한다.
  냉기의 딜 공백은 **분쇄(Shatter)** 메커니즘으로 해소 — 동상 스택 → 폭발, 냉기 계보
  스케일, 냉기 메인 덱(빙결사 다수 + 상인 경제) 성립 (타워스킬시트 §5 참조).
- 수렴 방지: 속사수 = 이중과세 하이 커밋 / 도트 = 방무 무과세 축(내장 메타 밸런서) /
  Q맵 함정 티어 + 중갑 웨이브. 새 요소 추가(아머 타입 2종화 등)는 플레이테스트에서
  수렴 관측 시의 예비 카드.

## 6. 경제 시스템 (v0.3 재편)

### 6.1 단일 골드 경제 (마나 삭제)
- 골드: 뽑기·지정 구매·상인 유지·계보 업그레이드·리롤·시민/선택권 구매를 모두 담당.
  마나와 골드 변환 도박은 삭제한다.
- 계보 업그레이드는 골드를 직접 소비한다. 비용은 계보별 레벨과 전체 계보 총합에 따라
  누진해, 상인 고점이 즉시 무제한 전투력으로 바뀌지 않게 한다.
- 재화 상한 몰수(1,000 초과 시 -100) 유지.
- 프로토타입 피드백: 후반 체감 문제의 주어는 "골드 고점/저점"보다 **소비 대역폭**.
  수입 분산을 키우는 것만으로는 해결되지 않으며, 후반에는 뽑기-배치-합성의 입력
  병목을 줄이는 별도 장치가 필요.

### 6.2 수입 스케줄 (드래프트 삭제 반영 — 수치 재검증 대기)
| 수입원 | 내용 | 성격 |
|---|---|---|
| 웨이브 클리어 | 라운드 비례 고정 보상 (기준식 100+10r, 재튜닝 대상) | 확정 바닥 |
| 상인 생산 | 고용·성장에 비례 (운영) | **운영 주도** |
| 개인 보스미션 | 쿨타임제, 단계별 보상, 빠를수록 이득 | **운영 주도** |
| 고정 미션 | 원소 컬렉터 / 정예 수집 / 신화의 증명 / 배짱 미션 | 지식층 |
| 보스 4회 | 택1: 골드 / 무료 계보 강화 / T3 선택권 + **무피해 격파 리베이트(+골드)** | 확정+선택 |
| 킬 업적 | 라운드 진행 비례 보너스 | 바닥 보강 |
| 히든 조합 미션 | 선택권 화폐 | 지식+운 |
- 드래프트 배수·스트릭 수입 삭제 → 수입 분산의 원천이 "베팅"에서
  "운영(상인·미션 타이밍)"으로 이동.

### 6.3 기존 게임과의 수입 구조 비교
- 랜타디: 웨이브 고정 + 쿨타임 개인미션 + 업적 체인 → **v0.3의 기본 골격** (회귀)
- Legion TD: 일꾼 경제 → 상인 고용제로 이식
- 스타2판 테라진 → 만능 토큰 / 원랜디 리롤 → 리롤

### 6.4 후반 소비 대역폭 해소 (프로토 피드백 반영)
- 원칙: 후반 골드 병목은 "수익곡선 실패"가 아니라 **저액 다회 소비의 입력 과밀**.
  해결 순서는 수입 재분산보다 소비 압축이 우선.
- **소비 속도 개선**:
  뽑기 `1회 -> 5회` 묶음, 빈 타일 자동 배치, 동일 타워 자동 합성(옵션형) 검토.
  후반의 재미를 판단이 아니라 탭 속도에 빼앗기지 않게 한다.
- **고액 소비처 추가**:
  저액 뽑기만으로 후반 자본을 태우지 못하므로, 200~500골드 단위의 선택형 소비처를
  연다. 예: 원소 후보 3택1, 상인 리롤, 시민 생성, 초월 파편 구매.
- **골드 -> 보정권 구매**:
  잉여 골드를 시민 조각·초월 파편·무료 계보 강화권으로 전환하는 후반 밸브를 둔다.
  고점 판의 재화가 "많이 벌었는데 못 쓰는 돈"으로 죽지 않게 한다.
- **수익 분산 조정은 후행**:
  소비 병목을 풀기 전에는 상인 고점/저점 스프레드를 더 키우지 않는다. 그렇지 않으면
  고점은 과잉 현금, 저점은 무행동으로 양극화된다.

## 7. 난이도 커브 — 클리어율 ≤1% (v0.4 GA 재베이스라인 확정)
- **밸런스 기준 격상 (방법론)**: "≤1%"의 주어는 손코딩 정책이 아니라 **유전 알고리즘이
  탐색한 최적 전략**. 정책 11파라미터(상인 킵, 계보 투자 비율, 예비금, 미션 마진,
  증강 선호 등)를 유전자로, 클리어율을 적합도로 진화시켜 착취 전략을 자동 발굴 —
  실제로 구 상수를 무효화하는 전략(골드 70~80% 계보 투자 + 81:19 이중 계보)을 GA가 찾아냈고,
  신 상수는 이를 이긴 값. (시뮬레이터: td_sim_v04_ga.py)
- 주의: 아래 GA 수치는 마나 삭제 전 v0.4 기준이다. 단일 골드 경제 반영 후
  `td_sim_v05`로 재베이스라인이 필요하다.
- **실측 (n=5,000)**: GA 최적 전략 R10 97% / R20 41% / R30 9.8% / **클리어 0.48%**
  (목표 분포 90+/40~55/10/≤1% 안착, 사망중위 R20). 평균 유저 프록시(비최적 운영)는
  R20 4%·클리어 0% — 최적과 평균의 격차 = 학습 곡선 (랜타디 포지션).
  완화 노브: HP 계수 1.285 ≈ 챔피언 0.8~1%.
- **확정 상수**: 몹 HP = 40 × **1.29**^r, R26+ ×1.06 가속. 아머 = 0.15 × 1.17^r
  (중갑 웨이브 ×4, 타격당 감산 — 공속 비례 피해). 웨이브 보상 = 100 + 10r.
  보스 ×1.7/1.8/2.0/3.4 + 골드 보상 + 무피해 리베이트 + v1 최소 증강 1택.
  상인 킬골드 사다리 1/1/2/3/5/8 (+황금 ×2), 라운드 상한 15+4r.
  특수 웨이브: 쇄도 R6/15/22/31 ×1.3(→1.1) / 신속 R8/17/27/35 ×1.4(→1.15) /
  중갑 R13/24/33 ×1.5(→1.2, 아머 ×4).
- GA 발견 텍스처: 극한 난이도일수록 최적 상인 수 감소(1.245에서 2기 → 1.29에서 1기,
  상인 = 여유의 사치재) / 개인미션 최적 마진 0.94 (안전선 아래 도전이 이득) /
  최적 플레이 기준 초월 5% = 20판에 1번의 잭팟.
- 업그레이드 곱연산(×1.03, 비용 20×(1+Lv/100)) / 초반 보호 유지.

## 13. 증강 시스템 — "슬롯은 v1부터, 풀은 단계적 확장" (확정)
결정: 나중 도입 시 비싼 것은 커브 재조정(시뮬로 저렴)이 아니라 **아키텍처**(규칙
변조를 위한 데이터 주도 설계)와 **게임 리듬**(보스 격파 3택1 의식)이므로,
시스템 슬롯은 v1에 넣고 콘텐츠 풀만 단계 확장한다.
- 타이밍: 보스 격파 시(R10/20/30) 증강 3장 제시 → 1택. 기존 보스 자원 보상 택1은
  증강 풀의 자원형으로 흡수, 무피해 리베이트는 별도 유지. 등급: 일반→희귀→전설.
- 핵심 원칙: **규칙 변조 우선, 스탯% 금지** (곱연산 경제 보호, 시뮬 이력 검증).

  | A. 빌드 확정형 | 최적해를 바꿈 (상인의왕, 원소 각성) | 100% + 커밋 보너스 | **덱 상태 기반 가중 제시**, 3장 중 최대 1장 |
  | B. 범용형 | 자원·소형 유틸, 바닥 올리기 | 90% (안전 세금) | 자연 등장 (보장 없음) |
  | C. 고점형 | 조건부·증강 간 시너지, 고분산 | 기대값 100% (달성 150~200%/실패 50%) | 희귀·전설 위주, 세트 후속은 선행 보유 시 등장 |

    "최소 보장" 대신 플레이어가 직접 제시를 고치는 방식 — 운영 철학 정합.

  - A형 가중 제시 = 함정을 "게임이 내 판을 알아봤다"로 전환. A형 1장 제한 +
    슬롯 리롤 = "3장 전부 빌드 강요" 차단. C형 고분산 = 초월·황금 상인과 나란한
    새 하이롤 축.

  [B] 자원형 골드 500 / 무료 계보 강화 / 정찰(다음 특수 웨이브 예고+완화) /
  황금 손(황금 변형 10→25%) / 혹한(감속 상한 60→70%) / 재정비(즉시 리롤 +2)

  킬골드 유지 — 상인 군단 덱 해금 (상인 3기+ 보유 시 등장률 가중)
  [C] 피의 계약: 오염 +15 감수 ↔ T4 선택권 (리스크 조건부)
