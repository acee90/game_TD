## 1. 한 줄 피치

스타크래프트 유즈맵 **"갓 타워 디펜스X"를 증강 드래프트 문법으로 재해석한
데스크탑 랜덤 합성 타워디펜스.**

## 2. 플레이어 판타지

"뽑기 운을 운영으로 이겨낸다" — 랜덤 조합이 만드는 타워 진영 위에,
내가 드래프트한 영웅 빌드가 판을 뒤집는 순간을 만든다.

## 3. 타깃 · 플랫폼 · 세션

- **타깃**: 유즈맵 세대(갓타디/랜타디 향수) + 오토배틀러·로그라이트 문법에 익숙한 PC 게이머
- **플랫폼**: 데스크탑(Unity). 웹(`web/`)은 밸런스 실험대이자 로직 원본.
  모바일은 장기 이식 후보 — 지금 결정하지 않는다.
- **세션**: 한 판 20~25분 목표 (현재 ~30분, 미달 — [명세](game-system-spec.md) 목표 지표 참조)
- **구조**: 무한 라운드 + 점수 + 명예의 전당. 런 단위 완결.

## 4. 감정 곡선 (한 판)

```
초반 (R1~15)   뽑기의 도박 — 조합이 터지느냐, 보스를 걸어볼 만하냐
중반 (R16~45)  엔진 조립 — 타워 진영 완성, 경제 배분, 빌드 방향 확정
후반 (R46~)    각성 — 시너지·잭팟이 터진 빌드가 벽에 부딪힐 때까지 달린다
```

## 5. 하지 않을 것

- **실시간 조작 요구** — 스킬은 자동 시전. 플레이어는 "무엇을 들지"만 고른다.
- **런 밖 메타 성장** — 판이 끝나면 점수만 남는다. 페이투윈 없음.
- **PvP** — TFT에서 증강은 가져오되 대전 구조는 가져오지 않는다.
- **원본의 맹목 재현** — EUD로 확인 불가한 부분을 추측으로 재현하지 않는다.

## 6. 레퍼런스

| 레퍼런스 | 가져오는 것 | 가져오지 않는 것 |
|---|---|---|
| 갓 타워 디펜스X (원본) | 경제 뼈대, 조합, 보스 소환, 십자 맵 | 미확인 EUD 로직의 추측 재현 |
| TFT | 증강 드래프트, 리롤, 등급 | PvP, 시즌 메타 |
| 뱀서라이크 | 빌드 몰입, 잭팟의 쾌감 | 조작 중심 이동 |

## 7. 코어 필러 — 모든 결정의 판정 기준

기능 목록이 아니라 **거절의 기준**이다. 새 기능·수치 변경은 "어느 필러를
강화하는가?"에 답해야 하고, 필러를 해치는 제안은 기각된다.

### P1. 조합의 도박이 기저 리듬이다
유닛 2기가 합쳐지는 순간(판당 ~70회)이 이 게임의 심장 박동이다.
이 리듬을 밋밋하게 만들거나(확정 조합) 노동으로 만드는(무한 리롤) 변경은 기각.
› 적용례: 리롤 상한 2회, 등급은 순수 뽑기 운.

### P2. 선택은 싱크의 경쟁에서 나온다
재화마다 소비처가 싸워야 결정이 생긴다. 싱크가 하나뿐인 재화는 자동 소비 =
선택 소멸이다.
› 적용례: 가스 3싱크(업글/리롤/스킬 개조), 프로브 지수 비용, 스탯 3트랙.

### P3. 타워는 기본기, 영웅은 이기는 수
타워가 판의 절반 이상을 책임지고, 영웅 몰빵의 상한은 딜 지분 ~50%다.
영웅이 혼자 캐리하면 TD가 아니고, 곁다리면 드래프트가 무의미하다.
› 적용례: 영웅몫 82% 과열 시 스탯 경제 조임, 체력 레벨 배수 분리.

### P4. 원본의 뼈대는 존중한다 — 단, 검증 가능한 것만
[원본확정] 수치는 함부로 바꾸지 않고, 확인 불가는 [프로토]로 명시하고 우리가
설계한다. 추측을 확정처럼 쓰지 않는다.
› 적용례: 시작 자원 55/6, 보스 보상 테이블, 킬 마일스톤은 원본 그대로.
