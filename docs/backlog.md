# 백로그 — 미확정 아이디어

> 상태: **현행** (수시 갱신) · 최종 갱신: 2026-07-10
> 명세에는 구현·결정된 것만 들어간다. 여기는 아이디어의 대기실이다 —
> 채택되면 명세로, 기각되면 사유와 함께 아래 "기각" 절로 옮긴다.

## 시스템 후보

- **빙고/미션 시스템** — 원본의 보유 미션류 (예: "티어1 유닛 전종 보유 시 골드").
  사용자 언급 있었으나 미착수.
- **초월 티어** — GOD 위 단계. 원본 §11.4의 `God LvNN` 49종이 힌트.
  진행 축이 R20에 소진되는 문제(도파민 리뷰 §5.1)의 후보 처방.
- **원거리 레인 보강** — 관통, 일제 사격 계열 증강 확률 상향.
  근거: 원거리 레인 최약 (R80, 영웅몫 19%).
- **중반 영웅 공백 처방** — R30~45에 영웅몫 0~10%. 후보 3안:
  중반 몹 HP↑·수↓ / 영웅 선공 어그로 / 보스 어그로 영웅 우선.
- **보스 리스크 확장** — 저지 불가로 리스크가 생겼지만 GA에서 보스 유전자
  신호가 여전히 약함. 보상·페널티 확대 검토.

## 기술 후보

- **Unity UI Toolkit HUD** — OnGUI 대체 (프레젠테이션 패스에서 보류).
- **Unity 명예의 전당 영속 저장** — 웹 localStorage 대응.
- **시트 업로드 자동화** — gen:balance 후 구글 시트 갱신 스크립트.
- **밸런스 CSV 확장** — 타워별 비용/DPS/역할, 적/웨이브 압력, 라운드별 수입·지출,
  합성 확률·티어 도달 분포 시트. (문서 리뷰 권고)
- **모바일 이식** — 장기. 조작 단순화가 전제.

## 프로세스 후보

- **플레이테스트 관찰 노트 템플릿** — 첫 외부 테스트 전에 준비.
  이해도/실패 원인/지루 구간/부당감 기록.

## 기각

- **증강 등급의 몹 체력 대가** — 구조적 함정으로 판명
  ([GA 검증](ga-policy-validation-v0.1.md) §3.4). 재도입하려면 비용이 강한 축에
  얹히지 않는 형태여야 한다.
- **영웅 타입 시작 선택 / Lv5 전직** — 정보 없는 선택 / 불필요한 관문.
  적응형 드래프트로 대체 (명세 §7).
- **리롤 무제한** — P1 위반 (노동화). 상한 2회로 채택됨.
