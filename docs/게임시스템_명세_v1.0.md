## 1. 맵과 경로

- **십자(cross) 지형**: 중앙 제단 타일 1 + 팔 4방향. 타워 타일 **41개**
  (십자 17 = 중앙 1 + 팔당 4 × 4방향, 모서리 4구역 × 6).
- **몹 경로**: 북측 왼쪽 입구에서 스폰 → 십자 외곽선을 **반시계** 일주 → 북측 오른쪽
  출구. 웨이포인트 14개, 경로 위 위치는 좌표가 아니라 "일주 거리" 스칼라 하나로 표현.
- **제단**: 게임 시작과 함께 십자 중앙에 무료 제공. 그 타일에는 타워 불가.
- 영웅과 몹은 경로 위로만 다닌다. 빈 곳 클릭 시 최근접 경로점으로 투영 이동.

## 2. 라운드와 웨이브

| 항목 | 값 | 출처 |
|---|---|---|
| 라운드 간격 | **22초 고정** (원본 57초) | [프로토] / [원본확정] |
| 첫 라운드 대기 | 5초 (원본 20초) | [프로토] |
| 웨이브 몹 수 | 12 + 3×(사이클 내 위치 0~4) = 12→24 | [프로토] |
| 스폰 간격 | 0.3초 | [프로토] |
| 라운드 수 | **무한** — 점수/명예의 전당으로 경쟁 | [프로토] |

**고정 간격인 이유**: 웨이브를 빨리 정리해도 다음 라운드가 앞당겨지지 않는다.
클리어 시 진행 방식이면 천천히 잡을수록 시간당 보스 소환 횟수가 늘어나는 악용이 생긴다.

### 2.1 난이도 곡선 — 5라운드 사이클

```
몹 HP    = 44 × 2.0^(사이클)         사이클 = floor((라운드-1)/5)
몹 수    = 12 + 3×(사이클 내 위치)
몹 장갑  = floor(라운드/5) × 3        (5라운드마다 계단)
몹 공격력 = 라운드에 선형 (근거리 접촉, 영웅·허수아비만 때린다)
```

- 사이클 안에서는 같은 몹, 수만 는다 — 예측 가능한 구간.
- 사이클이 넘어가면 HP가 2배로 뛴다 — "새 몹이 나오면 각도가 선다."
- 장기 성장률은 라운드당 ×1.149 지수와 같지만 플레이어는 구분선형으로 읽는다.
- 장갑은 타격당 감산(최소 10% 관통) — 저티어 유닛을 5라운드 주기로 무력화해
  "다음 계단 전에 티어를 올려라"는 압력을 만든다.
- **누출**: 몹이 출구에 도달하면 라이프 -1, 미네랄 +5 [원본확정]. 시작 라이프 20.

## 3. 유닛(타워)과 조합

- **생성**: 12 미네랄, 랜덤 유닛 (4종족 × 티어 풀, 7종 균등) [원본확정 비용/프로토 풀].
- **조합**: 같은 이름 2기가 모이면 **자동으로** 다음 티어 랜덤 유닛 1기로 합성.
  연쇄 조합 지원. 이것이 이 게임의 기저 도파민이다(판당 ~69회).
- **티어**: 5단계 (Lv1~4 + **GOD**). 티어 피해 배수 [1, 3, 9, 28, 95] × 기본 7.
  사거리 [120, 140, 160, 185, 225]. 기본 공격 간격 0.9초.
- **태그**: power(피해 1.6×, 사거리 1.15×, 보스 우선 타겟) / splash(피해 0.7×, 광역) /
  speed(간격 0.5×).
- **크리쳐 종족**: 보조 타워 — 피해 0.6×이지만 사거리 안 몹 감속.
  감속은 **가장 강한 것 하나만** 적용(도배 방지). 티어별 [0.9, 0.84, 0.76, 0.66, 0.5].
- **판매** 가능. GOD 풀은 보스 처치 수에 따라 조기/확장 분기.

## 4. 경제 — 재화마다 싱크가 경쟁한다

```
미네랄 → 유닛(12) / 프로브(30×1.3ⁿ, 최대 16기) / 영웅 스탯(25+14n)
가스   → 종족 업그레이드(8+4n+n²) / 증강 리롤(12→24, 최대 2회) / 스킬 개조(30×1.35ⁿ)
```

**설계 원칙**: 싱크가 하나뿐인 재화는 자동 소비가 되어 선택이 사라진다.
(개편 전 GA가 전 세대 "프로브 8 + 가스 즉시 업글"로 수렴했던 이유.)

### 4.1 소득 [원본확정 중심]

| 소득원 | 값 |
|---|---|
| 시작 자원 | 미네랄 55 · 가스 6 |
| 웨이브 클리어 | 14 + 3×라운드 [프로토 — 원본은 반복 20킬 보상] |
| 보스 처치 | Lv1~6: 5 / 8 / 13 / 20 / 29 / 39 |
| 킬 마일스톤 | 200킬 간격 20단계 (+5~15) |
| 누출 위로금 | +5 |
| 가스 | 프로브 1기당 0.25/초 |

### 4.2 프로브 딜레마

비용이 지수(30×1.3ⁿ)라 "몇 기까지"가 상황 의존적 결정이다.
측정: 프로브 4/8/14 → 사망R 90/93/90 — 수렴 없음(선택 성립).

### 4.3 종족 업그레이드

가스로 종족별 공격력 +10% 복리. 비용 8+4n+n². 타워 화력의 후반 축.

## 5. 보스 — 상시 소환, 리스크 있는 결정

- **소환**: 라운드와 무관하게 언제든, 쿨타임 45초, 비용 0.
- **해금**: Lv N 처치 → Lv N+1 개방. 열린 레벨은 계속 재소환 가능 (낮은 레벨 = 안전하지만 보상 적음).
- **HP**: 1150 × 2.15^(Lv-1) · **장갑**: 3×Lv · 속도 26.
- **보스는 영웅에게 저지되지 않는다** — 지나가며 칠 뿐. 화력이 모자라면 일주를 마치고
  **라이프 2+Lv를 문다**. 도발 인형(허수아비 개조)만 보스를 붙잡을 수 있다.
- **Lv1 기준선** (테스트로 고정): 시작 미네랄 55(유닛 4기)로는 **2단계 조합이 떠야**
  잡는 판(85%)이고 안 뜨면 가까스로 놓친다(15%). 75(6기)면 98%.
- 보스 처치는 영웅 XP의 페달이기도 하다 (xpPerBoss).

## 6. 영웅 — 세 개의 곱연산 축

```
영웅 파워 = 스탯(골드) × 레벨 배수(경험치) × 증강 배수(선택)
```

자원마다 축이 하나씩. 골드는 포인트를 사고, 경험치는 배수로 키우고, 증강은
그 위에 곱해지거나 특수능력을 얹는다.

### 6.1 스탯 (골드 구매)

| 스탯 | 효과 | 기본값 |
|---|---|---|
| 힘 | 공격력 +1/pt · 체력 +18/pt | 7 |
| 민첩 | 공격 속도 +4%/pt (간격 하한 0.25초) | 8 |
| 지능 | 스킬 피해 +3.5%/pt | 8 |

- 비용: 25 + 14×(그 스탯 구매 횟수) — 선형.
- **성장 곡선**: 20번째 구매마다 지급 포인트 +1 (1pt→2pt→3pt…).
  후반의 넘치는 골드가 점점 굵은 구매로 환전된다.
- 포인트당 효과는 선형이지만 레벨·증강 배수가 곱해져 후반 1포인트의 절대값이 더 크다.
- 빌드별 주스탯: 탱커=힘, 원거리=힘+민첩, 마법사=힘+지능.

### 6.2 레벨 (경험치)

- 공격력·체력의 **레벨 배수**: 공격력 ×(1+2.4×(L-1)), 체력 ×(1+1.0×(L-1)).
  체력 기울기를 따로 낮춘 이유: 같으면 힘몰빵 탱커가 부술 수 없는 벽이 된다.
- XP: 몹 1 / 영웅 막타 ×2 / 보스는 별도 보너스. 레벨업 시 완전 회복.
- 레벨업 비용: 14 × 1.06^L 지수 — 최고 레벨은 실전 ~47에서 멎는다.
- **초반 가드** (테스트 고정): Lv1 DPS < GOD 타워 5%, Lv9 < 25%, 무증강 Lv40 < 100%.

### 6.3 이동·어그로·부활

- 영웅은 경로 위로만 이동(속도 88). 어그로 범위 110 — 사거리 안 몹은 영웅 앞에
  멈춰 싸운다(모으기+지연). **보스는 예외**(§5).
- 붙잡힌 몹에게 타워가 넣는 피해는 `tankAssistDamage`로 집계 — 탱킹의 가치가 숫자로 보인다.
- 사망 시 12초 후 제단에서 부활(증강으로 단축 가능).

## 7. 증강 — 적응형 드래프트

영웅 타입/전직은 **없다**. 빌드 방향은 드래프트의 관성이 만든다.

- **스케줄**: 영웅 레벨 [9, 16, 24, 30, 35, 42], 이후 8레벨마다. 선택 중 게임 일시정지.
  80/20 — 앞의 4개는 거의 모든 판이 받고(진행률 절반 내), 5·6번째는 잘 버틴 판의 보상.
- **선택지**: 26종 풀에서 3장. **가중치 = 1 + 0.9 × (그 계열 보유 수)** — 몰수록 그
  계열이 더 잘 뜬다. 강제가 아니라 관성: 첫 장부터 특화해도, 범용부터 집어도 된다.
- **계열** 5종: 탱커 / 원거리 / 마법사 / 스탯 / 유틸.
- **등급**: 실버 1.0× (55%) / 골드 2.0× (33%) / 플래티넘 3.5× (12%). **대가 없음** —
  순수 뽑기 운. 높은 등급이 뜬 순간이 그 판의 도파민이다.
- **리롤**: 가스 12(2회째 24), **선택당 최대 2회**.
- **시너지**: 같은 계열 3개 = 특화, 5개 = 대특화. (예: 원거리 대특화 공격력 +100%
  공속 +30%.) 후반 역전은 레벨이 아니라 여기서 나온다.
- 제약: 스택 상한 / 대폭발은 충격파(광역) 필요 / 스킬은 하나만 / 스킬 개조는 보유 스킬만.

## 8. 액티브 스킬 — 자동 시전

플레이어는 **무엇을 들고 어떻게 개조할지만** 고른다. 시전은 조건 충족 시 자동.

| 스킬 | 효과 | 쿨 | 자동 시전 조건 |
|---|---|---|---|
| 소용돌이 | 주변 전체에 공격력 3× | 8초 | 2기 이상 |
| 일제 사격 | 사거리 안 4명에 각 2× | 7초 | 1기 이상 |
| 유성 | 최다 밀집점에 6× 광역 | 13초 | 3기 이상 |
| 허수아비 | 앞에 미끼 설치 (탱킹 대행) | 18초 | 앞쪽에 몹 |

- 스킬 피해 = 영웅 공격력 × 스킬 배수 × **skillPower(지능)** × 개조 배수.
- **개조 2트랙**: 증강(질 — 폭발 화살, 연사, 도발 인형 등 질적 변화) +
  가스 구매(양 — 피해 +8%/회, 쿨 -6%/회, 비용 30×1.35ⁿ).

## 9. 점수와 명예의 전당

- 라운드 통과: 60 × 1.15^라운드 (지수 — 사망 라운드 차이가 점수 격차로 증폭된다).
- 킬 10 / 영웅 레벨업 300 / GOD 타워 5,000 / 누출 시 라운드 점수의 25% 감점.
- 무한 라운드 + 로컬 명예의 전당(웹).

## 10. 검증된 밸런스 스냅샷 (8~10시드 중앙값)

| 운영 | 사망R | 영웅몫 | 탱킹어시 |
|---|---|---|---|
| 타워몰빵 | 84 | 5% | 17% |
| 마법사(힘+지능)·가스 업글 | 89 | 43% | 32% |
| 마법사·가스 스킬개조 | 86 | 48% | 30% |
| 탱커(힘몰빵) | 95 | 15% | 49% |
| 원거리(힘+민첩) | 80 | 19% | 26% |

- 최적 영웅 빌드가 판의 절반 남짓(43~48%)을 친다 — 타워도 절반을 유지.
- 탱커는 자기 딜 대신 어시스트(49%)로 기여 — 정체성 분리.
- 가스 배분 두 답(업글/스킬)이 3라운드 안쪽 — 선택 성립.

## 11. 설계 결정 기록 (왜 이렇게 됐나)

| 결정 | 이유 |
|---|---|
| 라운드 고정 간격 | 클리어 진행식은 보스 쿨 악용을 만든다 |
| 5라운드 사이클 난이도 | 지수를 구분선형으로 — 예측 가능 + 주기적 긴장 |
| 영웅 타입/전직 **삭제** | 정보 0의 시작 선택 → Lv5 전직 → 최종적으로 드래프트 관성으로. 선택이 맥락을 갖는 지점은 증강 선택 그 자체다 |
| 등급의 몹 체력 대가 **삭제** | 보너스는 약한 축(영웅), 비용은 강한 축(타워)에 얹혀 GA가 탐욕 0으로 수렴 — 구조적 함정이었다 |
| 골드 강화(×1.12) → 3스탯 구매 | 단일 트랙은 "사냐 마냐"뿐. 스탯 3종 + 성장 곡선으로 골드 운영에 빌드가 생겼다 |
| 체력 레벨 기울기 분리(1.0 < 2.4) | 힘몰빵 탱커가 부술 수 없는 벽(R114)이 됐다 |
| 보스 저지 불가 | 어그로 110 이후 무한 저지로 보스 실패 모드가 소멸 — 소환이 공짜 소득이었다 |
| 가스 싱크 3종 경쟁 | 프로브→가스→업글이 통째로 무뇌 루틴이었다 (GA 전 세대 수렴) |
| 리롤 상한 2회 | 무제한이면 플래티넘까지 굴리는 노동이 된다 |
| 스탯 비용 기울기 14 | 후반 포인트 총량 제한 — 영웅몫 82% 과열의 해법은 포인트 경제였다 (레벨 기울기 조정은 구매 포인트가 같은 곱을 타서 역효과) |

## 12. 미해결 과제

1. **원거리 레인 최약** (R80, 몫 19%) — 단일 대상이라 어그로가 모아준 뭉침을 못 쓴다.
   후보: 관통, 일제 사격 계열 증강 확률 상향.
2. **중반 영웅 공백** (R30~45 영웅몫 0~10%) — 타워 엔진 폭발기에 몹이 영웅까지 안 온다.
3. **탱커 생존 우위** (+11R) — 어시스트 정체성은 맞지만 격차 관찰 필요.
4. **판 길이** ~30분 (R85 × 22초) — 난이도 곡선과 묶어서.
5. **빙고/미션 시스템** (원본의 보유 미션류) — 미착수.

## 13. 밸런스 시트 관리 방침

- **원본은 코드다** (`web/src/data/*.ts`). 사람이 시트를 고치지 않는다.
- `npm run gen:balance` → `docs/balance/*.csv` 7종 자동 생성.
  `tests/balance-csv.test.ts`가 코드-시트 일치를 강제한다 (어긋나면 CI 실패).
- 외부 공유(구글 시트 등)는 **읽기 전용 사본**으로 올린다. 시트에서 수치를 바꾸고
  싶으면 코드를 바꾸고 재생성한다 — 역방향 동기화는 하지 않는다.
