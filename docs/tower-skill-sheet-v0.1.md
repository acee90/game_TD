## 0. 전역 규칙 (모든 수치의 파생 원천)

  `[1, 2.5, 7, 21, 70, 280]`. 티어 상승은 선형이 아니라 가속 커브다. 인접 티어
  배율은 대략 ×2.5 → ×2.8 → ×3.0 → ×3.33 → ×4.0으로 커진다.
- **계보 배율**: 해당 원소 타워 공격력은 `1.03^계보Lv`로 증가한다. 레벨당 상대
  증가는 +3%로 일정하지만, 누적값은 복리라 선형이 아니다.
- 아래 아키타입별 상세 표의 공격력 숫자는 스킬 컨셉/Q맵 초안이다. 프로토타입의 실제

  기본 랜덤 뽑기 단가는 낮게 유지하되, 상위 티어 실효 비용은 `동일 티어 2기` 규칙,
  랜덤 상위 결과, 슬롯 압박, 지정 구매 프리미엄으로 급격히 상승시킨다.

- 위 표는 실패·랜덤 결과·필드 공간 부족을 제외한 이론상 최저치다. 실제 특정 고티어를
  노리는 비용은 이보다 훨씬 높아야 하며, 지정 구매는 라운드와 사용 횟수에 따라
  비선형 프리미엄을 받는다.
- **유틸 예산 전환**: 유틸 아키타입의 자체 DPS = 예산의 35~50%. 나머지는 팀 증폭
  (감속=유효 처치시간, 방깎=팀 타격당 데미지, 스턴=유효 정지시간)으로 전환.
  기준: 표준 필드(딜러 6기 동반)에서 유틸 1기의 팀 기여 ≈ 동티어 딜러 1기.
- **방어 모델 = 타격당 감산**: 타격 데미지 = max(1, 공격력 − 방어력).
  → 방깎은 타격 횟수 비례 수혜(속사수 최대), 중갑은 저타수 고딜 우대. 단일 규칙에서
  배율 시너지와 중갑 웨이브의 근거가 모두 나옴.
- **CC 감쇠**: 감속 스택 상한 60% / 스턴 중 감속 무효 / 동일 대상 스턴 3초 내
  재적용 시 지속 반감.
- **성능계수(Q)**: 티어×아키타입별 0.7~1.4 의도 배치 (아래 표의 Q).
  Q<0.9 칸은 반드시 보조 가치 명시. 계수는 공격력(유틸은 특수 수치)에 곱해 적용.
- **서포트 스케일링 원칙 (체크박스 방지)**: 서포트 수치는 해당 원소 계보 레벨에
  연속 스케일한다 — 방깎량 = 기본 × (1 + 대지Lv×1%), 광역 밀집 계수 = 감속량 ×
  존 커버리지의 연속 함수 (냉기Lv의 반경 성장이 직결). 아머 커브(×1.17/R)가
  타격력 성장(≈×1.14/R)보다 빨라 미업글 서포트는 후반에 반드시 뒤처진다.
  의존 페어는 전부 **크로스 원소**(파쇄자·대지→속사수·번개 / 빙결사·냉기→화포·화염)
  이므로, 메인딜 덱 = 이중 계보 투자 강제 → 골드 투자 비율이 빌드 결정이 된다.
- **딜러 축 수렴 방지 3종**: ① 속사수 = 아머 이중과세(번개+대지 풀투자 요구,
  최고 천장·최악 골드 효율의 하이 커밋 빌드) ② 도트 = 방무 무과세 축 (아머
  인플레가 커질수록 상대 가치 자동 상승 — 내장 메타 밸런서) ③ Q맵 함정 티어 +
  중갑 웨이브 3회 = 속사수 헌팅 라운드. 예비 카드: 플레이테스트에서 수렴 관측 시
  아머 타입 2종화 검토 (지금은 복잡도 억제).
- **원소 이중 역할 원칙**: 모든 원소는 남의 파트너이자 자기 메인 플랜을 가진다 —
  번개 = 속사·스턴 템포 러시 / 대지 = 저격 처형 / 화염 = 화포·도트 소각 /
  냉기 = **분쇄 컨트롤 + 상인 경제**. "서포트 전용 원소" 계급은 존재하지 않는다.
- **부기능 계단 (v0.1.1 개정)**: T1 = 기본기. **T2~T3 = 시그니처 맛보기**(축소판,
  DPS 환산 5% 이내로 티어 예산에 포함) — 초반부터 아키타입의 손맛과 계보 선호가
  생기게. T4 = 시그니처 완성. T5 = 강화. T6(초월) = 필드급 궁극기.
- 사거리 단위: 칸(타일). 표준 3.0. / 공속: 초당 타격 수(APS).

## 10. 계보(원소) 업그레이드 — 골드 직접 소비처

- **소비**: 계보(원소) 단위 업그레이드. 골드를 직접 사용한다. 프로토 비용식은
  `50 + 선택계보Lv × 8 + 전체계보Lv합 × 2`. 같은 원소의 전 티어 타워가 공유한다.
- **설계 의도**: 계보 업그레이드는 후반 잉여 골드의 주요 소비처이자 곱연산 엔진이다.
  별도 변환 재화를 두지 않아 뽑기, 지정 구매, 계보 투자, 시민/선택권 구매가 같은
  골드를 두고 경쟁한다.
- **원소별 차등 효과** (기본은 모든 원소 피해 `×1.03^Lv`, 일부 원소는 추가 효과):

| 원소 | 업글 명칭 | 레벨당 효과 | 빌드 선언 |
|---|---|---|---|
| 화염 | 화력 | 피해 ×1.03 | 순수 화력 — 광역·도트 소각 빌드 |
| 번개 | 속도 | 피해 ×1.015 + 공속 ×1.015 | 템포 — 연사 가속 스택·스턴 시행 횟수 동반 상승 |
| 대지 | 파괴 | 피해 ×1.025 + 방깎량 +1% | 관통 — 중갑·보스 특화, 속사수 시너지 증폭 |
| 냉기 | 지배 | 피해 ×1.02 + 5레벨마다 감속존 반경 +3% · **분쇄 위력 +8%** · 상인 킬골드 +1 | 시간과 돈 — 분쇄 컨트롤·경제 빌드 |

- 설계 노트: 냉기는 감속 수치(상한 60%) 대신 **반경**을 성장시켜 CC 감쇠 규칙과
  무충돌. 번개 공속은 연사 가속·스턴·마비의 시행 빈도까지 올리는 다층 스탯.
  운영 기준선은 재시뮬 필요. 초안은 R6 이후 잉여 골드의 45~65%를 주력 계보에 직접 투자.
