# 머지 가디언즈 (가제) — 랜덤 합성 타워디펜스

랜타디(스타크래프트 랜덤 타워 디펜스) 계보를 잇는 모바일 우선 랜덤 합성 디펜스.
**고정 40라운드 · 클리어율 ≤1%의 극악 게이건틀릿 · 운이 겹치는 판만 정상에 도달.**

## 핵심 설계 요약

- **코어 루프**: 뽑기(8종 균등) → 같은 것 2개 = 상위 랜덤 합성 → 배치 → 고정 웨이브 방어
- **단일 골드 경제**: 골드로 뽑기·지정 구매·계보 업그레이드·후반 보정권을 모두 처리
- **상인**: 뽑기 등장(12.5%) + T1 전액 환급 — "획득은 운, 유지는 운영", 처치당 골드
- **의존 페어**: 속사수↔방깎(대지), 화포↔감속(냉기) — 서포트 수치가 계보 레벨에
  스케일하는 크로스 원소 이중 투자 구조
- **분쇄**: 냉기의 메인 딜 플랜 (동상 5스택 → 폭발) — 서포트 전용 원소 없음
- **초월(T6)**: 동일 신화 2개 = 확정. 최적 플레이 기준 20판에 1번의 잭팟
- **증강**: 보스 격파 시 3택1 (슬롯별 리롤 1회), v1 최소 풀 8종 → 확장 풀 로드맵

## 확정 밸런스 (GA v0.4 검증)

| 항목 | 값 |
|---|---|
| 몹 HP | 40 × 1.29^r (R26+ ×1.06 가속) |
| 아머 | 0.15 × 1.17^r (중갑 ×4, 타격당 감산) |
| 웨이브 보상 | 100 + 10r |
| 보스 배수 | ×1.7 / 1.8 / 2.0 / 3.4 |
| 검증 기준 | **유전 알고리즘이 찾은 최적 전략의 클리어율 ≤1%** (실측 0.48%) |

## 프로토타입 실행

`prototype/index.html`을 브라우저로 열면 됩니다. 검증 대상: 합성 손맛,
상인 킵/매각 판단, 골드 직접 계보 투자, 특수 웨이브의 소프트 체크 체감.
(증강·미션·영토 확장·보스 기믹은 프로토 범위 외 — 디자인 문서 참조)

주의: 프로토타입은 합성 손맛 체험을 위해 **문서 확정치 대비 완화된 체험 모드**
(보상 160+16r · 시작 300골드 · 뽑기 40골드 · 타일 24개 · HP 계수 1.22).
출시 밸런스는 문서 §7의 GA 확정치(뽑기 60 · HP 1.29)를 따른다.

```bash

npm test        # 게임 로직 스모크 테스트
npm run build   # 타입체크 + 프로덕션 빌드
```
